EduQuiz Laravel API Web Version - Starter Kit
---------------------------------------------

INSTRUCTIONS FOR YOUR OTHER LAPTOP:

1. Apne dusre laptop par terminal/command prompt kholein aur ek naya Laravel project banayein:
   > composer create-project laravel/laravel eduquiz_web

2. Jab project ban jaye, toh is ZIP file ko extract karein. Is mein jo folders hain (app, resources, routes), unko copy kar ke apne naye 'eduquiz_web' folder mein paste kar dein (Replace file prompts ko Yes kar dein).

3. Project ke andar ja kar server start karein:
   > cd eduquiz_web
   > php artisan serve

4. Browser mein jayein: http://127.0.0.1:8000

Mubarak ho! Aapka Laravel frontend aapke bgnuf22eight.com wale APIs ke sath connect ho gaya hai. 
Isme Login, API Token Session Management, aur Teacher/Student Dashboards shamil hain!
